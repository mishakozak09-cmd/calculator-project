# Простий калькулятор

Консольний калькулятор для виконання базових арифметичних операцій.  
Навчальний проєкт для вивчення Git, GitHub та Python.

## Опис

Цей проєкт представляє собою простий калькулятор з текстовим інтерфейсом,  
який дозволяє виконувати основні математичні операції в інтерактивному режимі.

## Функціональність

- Додавання двох чисел
- Віднімання двох чисел
- Множення двох чисел
- Ділення двох чисел (з перевіркою ділення на нуль)
- Піднесення до степеня (`**`)
- Обчислення остачі від ділення (операція `%`)

## Вимоги

- Python 3.7 або вище
- Операційна система: Windows / macOS / Linux


Видали її й встав замість ОЦЕ (повністю):


## Встановлення

1. Клонуйте репозиторій:
```
git clone https://github.com/mishakozak09-cmd/calculator-project.git
```
2. Перейдіть у папку проєкту:
   
cd calculator-project

3. Запустіть програму:
   
python calculator.py


## Використання

Після запуску програми:

Оберіть операцію: +, -, *, /, **, %

Введіть перше число

Введіть друге число

Отримайте результат

## Приклад роботи

=== Простий калькулятор ===
Операції: +, -, *, /, **, %
Для виходу введіть 'exit'

Введіть операцію (+, -, *, /, **, %) або 'exit': +
Введіть перше число: 15
Введіть друге число: 7
Результат: 22.0

Введіть операцію (+, -, *, /, **, %) або 'exit': **
Введіть перше число: 2
Введіть друге число: 8
Результат: 256.0
